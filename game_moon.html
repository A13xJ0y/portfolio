Core Gameplay Loop

Moon Pioneer - небольшой прототип - клон базовых механик игры про распределение ресурсов Moon Pioneer. В прототипе есть фабрики трех видов: четыре красных, одна синяя и одна фиолетовая. Фабрики производят так называемый "Продукт" в виде шариков, которые игрок, управляющий белой капсулой может подбирать в рюкзак позади капсулы, когда подходит к контейнерам фабрики.

У всех фабрик кроме красной есть два контейнера - контейнер для требуемых продуктов и контейнер для произведенных продуктов. С разной скоростью, фабрики создают продукты соответствующего цвета. Для создания одного синего продукта необходимо два красных. Для создания одного фиолетового продукта необходим один красный и один синий продукт.

Игрок управляет перемещением капсулы джойстиком, который появляется в месте нажатия на экран. Белая кнопка справа перезагружает игру и опустошает рюкзак.
